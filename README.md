# Calculator

**A simple, responsive calculator web app** that performs basic arithmetic operations.

**Live Demo:** https://sharmayush06.github.io/Calculator/

---

## About

This is a browser-based **Calculator application** built with HTML, CSS, and JavaScript. It lets users perform basic mathematical operations in a clean and intuitive interface.

---

## Features

- Add, subtract, multiply, and divide  
- Clear and reset input  
- Responsive layout  
- Works in modern browsers

---

## Live Preview

You can use the calculator instantly — just open the **Live Demo** link above in your web browser.

---

## Built With

This project is built using:

- HTML  
- CSS  
- JavaScript

No backend — all logic runs in the browser.
